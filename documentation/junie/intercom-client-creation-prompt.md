Create clients inside the "org.mojave.core.wallet.intercom.client" package for
the **Controllers in "org.mojave.core.wallet.intercom.controller" package.
You can refer to the implementation here in "org.mojave.core.participant.intercom.client" as a sample.
You also need to implement the "org.mojave.core.wallet.intercom.client.WalletIntercomClientConfiguration".

You must follow the rules below:
- Use proper naming conventions for classes, methods, and variables.
- Avoid using FQNs whenever importing the classes. Must use the "import" statement.
- Ensure that the client handles all the methods in the controller.
- Follow the coding style and conventions used in the existing codebase.
